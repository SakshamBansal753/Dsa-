- 🔹 Graphs (BFS, DFS, Dijkstra, etc.)

- 🔹 Sorting & Searching Algorithms  
- 🔹 Greedy, Dynamic Programming, and Bit Manipulation  
- 🔹 Sliding Window, Two Pointer, and other Patterns  
- 🔹 LeetCode, GFG, InterviewBit, and Contest Problems
